## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic knowledge of HTML/CSS/JavaScript (for customization)

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/Kapil-Iyer/kapil-iyer-portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd kapil-iyer-portfolio
   ```

3. **Open in your browser**
   - Double-click `index.html` to open locally
   - Or use a local server for development

### Local Development
For the best development experience, use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```

## 🎨 Customization

### Colors
The portfolio uses CSS custom properties for easy theming:
```css
:root {
    --primary: #4f46e5;      /* Main brand color */
    --accent: #10b981;       /* Accent color */
    --dark: #0f172a;         /* Dark background */
    --light: #f1f5f9;        /* Light text */
}
```

### Content
- **Personal Information**: Update `index.html` with your details
- **Experience**: Modify `data/experience.json` for your work history
- **Projects**: Edit `data/projects.json` to showcase your work
- **Skills**: Update the skills section in `index.html`

### Styling
- **Typography**: Modify font sizes and weights in CSS
- **Layout**: Adjust spacing and grid layouts
- **Animations**: Customize transition timings and effects
